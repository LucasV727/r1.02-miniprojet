Situations d’apprentissage et d’évaluation

SAÉ S1.02 : Comparaison d’approches algorithmiques

Problématique : Choisir une approche de résolution pour un problème donné.

Description : Implémenter et comparer plusieurs approches, puis mesurer leurs performances. Livrables : code et présentation comparative.

Cursus : 23h (12h projet tutoré + 11h formation complémentaire).

SAÉ S2.02 : Exploration algorithmique d’un problème

Problématique : Approfondir les stratégies de résolution durant le développement.

Description : Explorer des solutions algorithmiques bas et haut niveau, analyser chemins d’exécution, structures de données et benchmarks.

Cursus : 30h (16h projet tutoré + 14h formation complémentaire).
