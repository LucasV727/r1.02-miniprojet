Compétence 4 : Gérer des données

Apprentissages critiques – Niveau 1

Concevoir et mettre en place une base de données à partir d’un cahier des charges client.

  • AC 1 : Mettre à jour et interroger une base de données relationnelle.
  • AC 2 : Visualiser des données.
  • AC 3 : Concevoir une base de données relationnelle à partir d’un cahier des charges.

Coefficients des ressources

Ressources R1.05 R1.06 R1.09

Situations d’apprentissage et d’évaluation

SAÉ S1.04 : Création d’une base de données

Problématique : Formaliser et implémenter une base répondant à un besoin client.

Description : Étudier, modéliser puis réaliser la base. Livrables : jeu de données, modèle, rapport et script de création.

Cursus : 23h (12h projet tutoré + 11h formation complémentaire).

SAÉ S2.04 : Exploitation d’une base de données

Problématique : Alimenter et exploiter une base pour répondre aux besoins métiers.

Description : Réaliser la base, préparer un jeu d’essais et proposer une visualisation pertinente des informations.

Cursus : 28h (16h projet tutoré + 12h formation complémentaire).

R1.05

Introduction aux bases de données et SQL

AC liés : AC 1, AC 3

Modélisation conceptuelle et relationnelle, algèbre relationnelle, bases du SQL et usage d’un SGBD.

R1.09

Introduction à l’économie durable et numérique

AC liés : AC 2

Fondements économiques, écoconception des services numériques et enjeux économiques des données.

R2.06

Exploitation d’une base de données

AC liés : Tous les AC

SQL avancé, visualisation de données, administration de SGBD (utilisateurs, rôles, droits) et sécurité.

R2.08

Outils numériques pour les statistiques descriptives

AC liés : AC 2

Analyse et représentation des données : paramètres de position, dispersion et visualisations (diagrammes, ajustements).
