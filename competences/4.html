Situations d’apprentissage et d’évaluation

SAÉ S1.04 : Création d’une base de données

Problématique : Formaliser et implémenter une base répondant à un besoin client.

Description : Étudier, modéliser puis réaliser la base. Livrables : jeu de données, modèle, rapport et script de création.

Cursus : 23h (12h projet tutoré + 11h formation complémentaire).

SAÉ S2.04 : Exploitation d’une base de données

Problématique : Alimenter et exploiter une base pour répondre aux besoins métiers.

Description : Réaliser la base, préparer un jeu d’essais et proposer une visualisation pertinente des informations.

Cursus : 28h (16h projet tutoré + 12h formation complémentaire).
