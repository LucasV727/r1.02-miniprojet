Situations d’apprentissage et d’évaluation

SAÉ S1.05 : Recueil de besoins

Problématique : Conduire un projet à partir d’un besoin client.

Description : Dialoguer avec le client pour formaliser ses attentes. Livrables possibles : questionnaire, maquettes, scénarios et présentation orale.

Cursus : 18h (12h projet tutoré + 6h formation complémentaire).

SAÉ S2.05 : Gestion d’un projet

Problématique : Piloter un projet simple de bout en bout.

Description : Analyser les besoins, rédiger un cahier des charges et produire un dossier de gestion (Gantt, Pert, budget…).

Cursus : 26h (16h projet tutoré + 10h formation complémentaire).
